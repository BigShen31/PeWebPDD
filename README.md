# 拼豆图片转换器 PeWeb

一个面向拼豆手工爱好者的静态网站：上传图片后自动生成适合拼豆制作的超清点阵图纸，可调尺寸、颜色数量、对比度，并支持导出高清 PNG。

## 亮点功能
- 密码访问控制：发货时生成一次性密码，用户需要输入后方可访问
- 超清拼豆图：可输出至 160×160 格尺寸，自动适配拼豆规格
- 色号标注：像素图纸格内显示色号，导出的 PNG 同样保留色号、网格与坐标
- 镜像翻转：支持左右镜像，适合布板方向或对称场景
- 色号替换：内置 32 种拼豆色卡，支持批量替换、叠加多组映射
- 图片上传与拖拽、实时预览、高清 PNG 导出

## 密码发放与配置
1. 打开 `password.json`，将 `passwordHash` 替换为你提供给用户的密码的 SHA-256 哈希
   - 可使用命令：`echo -n "实际密码" | sha256sum`
   - 只保留 64 位十六进制字符串
2. 将生成的密码明文通过订单或客服发送给用户
3. 用户输入正确密码后会缓存验证结果（浏览器 `localStorage`），刷新仍可使用；如需失效，更新 `password.json` 并重新部署

## 本地运行
```bash
npm install
npm run dev
```

## 部署到 Vercel
1. 将本目录作为项目根目录
2. 在 Vercel 创建新项目并导入该目录
3. 框架选择 `Other`
4. 构建命令留空或使用 `npm run build`
5. 输出目录留空（使用根目录静态文件）

## 文件说明
- `index.html`：页面结构、密码门禁和控制面板
- `style.css`：页面样式、色号面板布局
- `app.js`：密码验证、像素化、色号匹配、镜像与替换逻辑、高清导出
- `password.json`：存储密码哈希
- `package.json`：项目脚本

## 使用说明
1. 输入访问密码，通过验证后进入工具
2. 上传图片（支持拖拽），或使用默认示例图案体验
3. 调整图纸宽度、颜色数量、饱和度、对比度等参数
4. 勾选“色块内显示色号”“镜像图案”等选项，必要时添加色号替换
5. 预览实时生成的点阵图；右侧统计展示每种色号的颗数
6. 点击“导出 PNG 图纸”获取高清图纸，内容包含网格、色号、数量统计

## 备注
- 密码校验仅在前端执行，请定期更换密码并更新 `password.json`
- 如需更高安全性，可配合 CDN/反向代理做访问限制
- 需要更完整的色卡，可扩展 `palette` 数组，新增色号即自动出现在替换选项中
