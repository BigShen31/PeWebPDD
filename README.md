# 拼豆图片转换器 PeWeb

一个面向拼豆手工爱好者的静态网站：上传图片后自动生成适合拼豆制作的点阵图纸，可调尺寸、颜色数量、对比度，并支持导出高清 PNG。

## 功能
- 图片上传与拖拽上传
- 自动像素化与拼豆颜色映射
- 网格显示与编号
- 颜色统计
- 高清 PNG 图纸导出
- 适配桌面与平板端

## 本地运行
```bash
npm install
npm run dev
```

## 部署到 Vercel
1. 将本目录作为项目根目录
2. 在 Vercel 创建新项目并导入该目录
3. 框架选择 `Other`
4. 构建命令留空或使用 `npm run build`
5. 输出目录留空

## 文件说明
- `index.html`：页面结构
- `style.css`：页面样式
- `app.js`：图片处理与导出逻辑
- `package.json`：项目脚本

## 使用说明
1. 上传一张图片
2. 调整图纸宽度、颜色数量等参数
3. 预览生成结果
4. 点击导出 PNG 图纸

## 备注
当前版本是纯前端静态实现，部署后可直接使用。
