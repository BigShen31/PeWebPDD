# 拼豆图片转换器 PeWeb

一个面向拼豆手工爱好者的静态网站：上传图片后自动生成适合拼豆制作的点阵图纸，可调尺寸、颜色数量、对比度，并支持导出高清 PNG。

## 功能
- 安全访问：发货时生成一次性密码，用户需先通过密码校验才能进入站点
- 图片上传与拖拽上传
- 自动像素化与拼豆颜色映射
- 网格显示与编号
- 颜色统计
- 高清 PNG 图纸导出
- 适配桌面与平板端

## 密码发放与配置
1. 打开 `password.json`，替换 `passwordHash` 为你提供给用户的密码的 SHA-256 哈希
   - 可使用命令：`echo -n "实际密码" | sha256sum`
   - 只保留 64 位十六进制字符串
2. 将生成的密码明文通过订单系统或客服消息发给用户
3. 用户输入正确密码后会自动缓存验证结果（`localStorage`），刷新仍可使用

## 本地运行
```bash
npm install
npm run dev
```

## 部署到 Vercel
1. 将本目录作为项目根目录
2. 在 Vercel 创建新项目并导入该目录
3. 框架选择 `Other`
4. 构建命令留空或使用 `npm run build`
5. 输出目录留空（使用根目录静态文件）

## 文件说明
- `index.html`：页面结构与访问门禁入口
- `style.css`：页面样式
- `app.js`：密码验证、图片处理与导出逻辑
- `password.json`：存储密码哈希
- `package.json`：项目脚本

## 使用说明
1. 用户在站点入口输入密码
2. 上传一张图片
3. 调整图纸宽度、颜色数量等参数
4. 预览生成结果
5. 点击导出 PNG 图纸

## 备注
- 密码校验在客户端完成，请定期更换密码并更新 `password.json`
- 如需更高级别安全（如请求限速、IP 白名单），建议在 CDN/反向代理层配置
